# ⚡ Al Qaswa – Wholesale Laptop Chargers Website

Al Qaswa is an animated, responsive wholesale website for laptop chargers featuring WhatsApp integration, live Google Maps, HP/Lenovo/Dell/Apple brands, and bulk inquiry support for B2B buyers in Sharjah.

## 📋 Features

- ✅ Fully responsive design (Desktop, Tablet, Mobile)
- ✅ Smooth animations with AOS library + floating particles
- ✅ Fixed WhatsApp button for instant bulk orders
- ✅ Live Google Maps embed of Sharjah Industrial Area 6
- ✅ Brand showcase: HP, Lenovo, Dell, Apple, ASUS, Acer
- ✅ Contact section with phone & address
- ✅ Smooth scrolling navigation
